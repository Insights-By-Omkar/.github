## How it's built

One human. The full stack — product strategy, system design, code, ops, support, billing, growth.

- **Engine work** is first-principles. No third-party astrology library. Just JPL ephemerides, Meeus, and tests.
- **AI architecture** is multi-tier — retrieval, online + offline evaluation pipelines, taste auditing across providers, structured tool use.
- **Operations** is dual-rail — Stripe + PayPal with chargeback defense, multi-region deploy, structured observability via Bodha.
- **Discipline** is versioned releases, real CHANGELOGs, and no fake "team" voice. The work speaks.
